## 🚀 SkyBurner Ultimate — Flagship Game

> **SkyBurner Ultimate** is a high-octane, vertical-scrolling arcade space-shooter built **entirely in Python** — no external assets required. Developed by **PCVR Studios** and powered by the custom **Atlas Nexus Engine**, the game runs on **Pythonista 3** for iPhone and iPad, delivering stunning vector-art visuals at full frame-rate on any modern iOS device.

### 🐍 100% Python — Three Clean Files, One Epic Game

Three Python files (`atlas_nexus.py`, `entities.py`, `skyburner.py`) power the entire experience — a testament to what Python can achieve on mobile. Every sprite, explosion, and star in the game is drawn using Python's built-in modules; there are **zero external assets**.

### 🎮 Features

| Feature | Details |
|---|---|
| 🔫 **4 Weapon Tiers** | Single laser → Dual cannon → Triple spread → Homing missiles |
| 👾 **3 Enemy Types** | Fighter, Cruiser, and Boss (appears every 5th wave) |
| 💥 **Multi-Phase Bosses** | Entry → Attack 1 → Attack 2 → Rage mode |
| ⚡ **Combo System** | Chain kills to build streaks and boost your multiplier up to **×8** |
| 🛡️ **Power-Ups** | Weapon upgrade, shield boost, health pack, extra bomb, speed burst |
| 🌌 **115-Star Parallax Starfield** | Gorgeous scrolling space backdrop for full immersion |
| 📊 **Full HUD** | Score, high-score, health bar, shield bar, lives, bomb count, wave indicator, boss health bar |
| 🎯 **Touch Controls** | Drag to move your ship, tap to drop a screen-clearing bomb — auto-fire lets you focus on dodging |
